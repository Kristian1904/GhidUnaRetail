Ghid Build Android

1)  Setările necesare

-        MSVC: MSVC 2019 (32bit)

 

-        Windows Kit: Windows 10 SDK

-        JDK: JDK 8

 

-        Android build-tools: 31.0.0

-        Android SDK Platform: android-31

 

-        ABI setat: armeabi-v7a



 

2)  Release build

-        Setează build pe Release

n caz contrar daca nu apare la Build Release, atunci: Project → Build Settings →Edit build configuration: Add Release → Seteaza un nume (ex. ReleaseA):

 

-        Dai release Build la Android: dupa care s-a creat acest .apk

-        Intră n folder-ul unde s-a creat acel apk, deschide un terminal, și scrie după exemplul tău de path:


nainte de a scrie comanda salvează si key-ul intr-un folder

D:\UNARetail-master\UNARetail-master\uaMobi\build\Android_Qt_5_15_2_Clang_Multi_Abi-ReleaseA\android-build\build\outputs\apk\release>D:\ExtensieAndroidd\build-tools\31.0.0\apksigner.bat sign ^

More? --ks Folderul unde ai creat key(Path)\myapp-release-key.jks ^

More? --ks-pass pass:unamia ^

More? --key-pass pass:unamia ^

si deja am intr-un folder nou acel apk + idsig

 

 

3)  Instalare pe device

 

-        Conectează device-ul Android

-        Asigură-te că fișierul server.py este n același folder cu .apk

-        Deschide un terminal n acel folder și rulează:
python server.py

-        Pe telefon, deschide browserul (Google Chrome) și tastează adresa IP afișată (ex: http://192.168.0.202:8000)

-        Vei vedea fișierul .apk listat. Dă click pe el și instalează aplicația.
